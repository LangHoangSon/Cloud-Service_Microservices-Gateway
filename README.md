# Cloud-Service_Microservices-Gateway
Hệ thống thương mại điện tử (E-commerce) hiện đại đòi hỏi khả năng mở rộng linh hoạt, xử lý hiệu quả và dễ dàng bảo trì trong môi trường có lưu lượng truy cập lớn. Tuy nhiên, kiến trúc nguyên khối (Monolithic) truyền thống thường gặp nhiều hạn chế như khó mở rộng, khó quản lý và dễ gây ảnh hưởng toàn hệ thống khi xảy ra lỗi.
